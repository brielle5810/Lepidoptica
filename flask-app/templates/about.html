About Our Project

This is a tool to assist in digitization efforts for University of Florida researchers transcribing butterfly specimens for the Lepidoptera collection.

Users can upload batches of images and our website will attempt to transcribe their text content using machine learning. Next, users can edit the transcriptions from the web interface and save them as a .csv file for bulk uploading to the collection database.

Designed by students Alexia Rangel Krashenitsa, Alex Harvey, Brielle Stokes, and Katja Karoleski at the University of Florida for Senior Project Spring 2025.

For our project repository and more information about our project, please visit
